# React + TypeScript + Vite

هذا المشروع جاهز للنشر على GitHub Pages.

## نشر تلقائي على GitHub Pages

1. ارفع المشروع إلى GitHub.
2. افتح Repository Settings > Pages.
3. اختر Source: GitHub Actions.
4. ادفع التغييرات إلى الفرع main أو master.
5. سيبدأ workflow التلقائي في بناء النشر وإكماله.

## تشغيل محليًا

```bash
npm install
npm run dev
```

## بناء الإنتاج

```bash
npm run build
```
